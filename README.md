🎮 The Concept
Classic snake. Simple mechanics. One twist:

🍎 Eat ten apples... and you'll spawn a super intelligent enemy that's coming for you.
Not just another snake. This AI predator uses pathfinding to hunt you down.
It calculates. It predicts. It remembers where you are.

The first nine apples? Just practice.
Apple number ten? The hunt begins. 👁️

⚔️ Gameplay Features
Feature	What it does
🎯 Smooth snake controls	No reversing into yourself like some kind of amateur
🧠 Dynamic difficulty	The AI hunts you using Manhattan distance pathfinding (A*-lite)
⚡ Tension building	The game changes completely after those ten apples
💀 Permadeath	One mistake and it's over. The AI doesn't miss.

📦 Installation & Running
1️⃣ Create a .py file
Copy the code into a file called snake_evolution.py

2️⃣ Open PowerShell (or terminal)
Navigate to where you saved the file:
powershell
cd C:\Users\YourName\Desktop
3️⃣ Run the game
powershell
python snake_evolution.py
✅ Requirements: Python 3.x installed (tkinter comes with Python)
